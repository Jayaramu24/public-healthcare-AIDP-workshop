Introduction

About this Workshop

Welcome to Public Healthcare AIDP Workshop. This workshop helps you build a healthcare lakehouse analytics flow on Oracle Cloud using Oracle AI Data Platform, Autonomous AI Lakehouse, and Oracle Analytics Cloud.

You will start from public-health and claims source data, refine it through Bronze and Silver medallion layers, publish a Claims star schema to AI Lakehouse, and use OAC to analyze service pressure, denial hotspots, funding gaps, and operational patterns.

Estimated Workshop Time: 3 hours for the core labs

Objectives

  • Understand the healthcare lakehouse architecture in Oracle Cloud
  • Build Bronze and Silver layers in AIDP
  • Publish a Claims star schema to Autonomous AI Lakehouse
  • Create an OAC dashboard with filters, map context, and assistant prompts

Lab Sequence

  • Lab 1: Set Up Core Resources
  • Lab 2: Build Bronze and Silver
  • Lab 3: Publish Claims Star Schema
  • Lab 4: Analyze in OAC

Roles and responsibilities

This page exists to remove the most common beginner confusion: who does what, and when. In the LiveLabs-style version, participants see the admin setup expectation before the labs start rather than discovering it inside the middle of a task.

Admin / facilitator
Provision shared AIDP, AI Lakehouse, OAC, Object Storage, users, catalogs, and starter assets.
Participant
Sign in to the prepared environment, run notebooks, validate outputs, and complete the guided OAC flow.

Required services

  • Oracle Object Storage
  • Oracle AI Data Platform with Spark
  • Autonomous AI Lakehouse
  • Oracle Analytics Cloud

Execution assets

Participants should start with the guide, the raw source bundle, and the Bronze/Silver starter notebooks. That is enough to begin without seeing the full asset inventory up front.
