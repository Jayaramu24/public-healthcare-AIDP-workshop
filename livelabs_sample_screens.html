Introduction

About this Workshop

Welcome to Public Healthcare AIDP Workshop. This guided workshop helps you build a modern healthcare analytics pipeline on Oracle Cloud using Oracle AI Data Platform, Autonomous AI Lakehouse, and Oracle Analytics Cloud.

You will work through the full lakehouse path for Metro Public Health Authority, starting from raw public-health and claims inputs, refining them through Bronze and Silver layers, publishing a Claims star schema to AI Lakehouse, and visualizing the guided business story in OAC.

Estimated Workshop Time: 3 hours for the core labs

Objectives

  • Understand the healthcare lakehouse architecture in Oracle Cloud
  • Build Bronze and Silver layers in AIDP from raw healthcare sources
  • Publish the Claims star schema to Autonomous AI Lakehouse
  • Create a guided OAC dashboard with filters, maps, and assistant prompts

Lab Sequence

  • Lab 1: Set Up Core Resources
  • Lab 2: Build Bronze and Silver
  • Lab 3: Publish Claims Star Schema
  • Lab 4: Analyze in OAC
